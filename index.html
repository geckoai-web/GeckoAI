Platform Capabilities

一套平台，串起資料製備到工業檢測決策。

01

Label Workspace

提供一般框與旋轉 OBB 標註、類別管理、Undo/Redo、多選編輯與模型輔助標註。

02

Detect Workspace

支援批次檢測、Golden Compare、背景切割、Fail Reason 與每張圖像的視覺證據輸出。

03

Training Workflow

支援 YOLO / YOLO-OBB 訓練、CPU/GPU Runtime 選擇、進度監控與封裝部署策略。

04

Automation Ready CLI

可整合 MES、排程器或產線腳本，支援 Watch Mode、JSON Summary 與 Exit Code 控制。

Workflow

專為實際工廠與 QA 現場設計的產品節奏。

Step 1

匯入影像與資料集

支援圖片資料夾、YOLO Dataset、Golden Package 與相機擷取來源。

Step 2

標註與建立標準

快速建立類別、調整 OBB、導出 Golden Package，形成可持續迭代的資料基礎。

Step 3

訓練與部署模型

依資料格式自動對應 YOLO 或 YOLO-OBB，並保留 CPU/GPU 相容彈性。

Step 4

檢測、比對、輸出報表

執行 Golden Compare、OCR Traceability 與 Dashboard Report，讓決策與追溯更清楚。

Use Cases

同一套產品，可落地在不同產業的視覺檢測任務。

咖啡包裝檢查

驗證包裝完整性、貼標位置、封口與閥件是否符合標準。

  • Golden Compare 做版型一致性比對
  • PASS / FAIL 理由可直接進入報表

咖啡豆分級

針對碎豆、黑豆、蟲蛀與異物做分類計數，建立品質數據資產。

  • 模型檢測 + 類別統計
  • 適合品保與採購決策輔助

PCBA 缺陷檢測

結合背景切割、Piece 檢測、Golden Compare 與 OCR，鎖定缺件與偏移問題。

  • 支援 board-level 與 piece-level 分析
  • 可將 ID / Sub-ID 寫入追溯報表
